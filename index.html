WebGL Game Engine

By Max Kerscher-Santelli

This is a from scratch 3D graphics renderer I have been working on in my spare time. I chose this project to learn more about the GPU and how developers can take advantage of it. To get close to the GPU I have chosen to work with just WebGL. WebGL is the API your browser uses to comunicate with the GPU. Thanks for checking it out.
You can view the source code at this Github Repo. Most of the work you can find under scripts/main.js. My shaders are in index.html.
Current Features:
  • Render Geometry
  • Textures
  • moveable camera
Upcoming Features
  • Light Sources
  • Scene Graph
  • More Complex Shaders
  • Interactivity
